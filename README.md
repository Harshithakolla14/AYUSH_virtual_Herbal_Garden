# AYUSH_virtual_Herbal_Garden

# Project Overview

AYUSH Virtual Herbal Garden is an AI-powered educational platform that combines traditional medicinal plant knowledge with modern technology. The system enables users to explore medicinal plants in a virtual environment, identify plants using AI, and receive health-related recommendations based on AYUSH principles.

# Features
🌱 Interactive Virtual Herbal Garden
🤖 AI-Based Medicinal Plant Identification
🔍 Plant Information and Benefits
🩺 Ayurvedic Remedy Recommendations
🧘 Yoga Posture Suggestions
🎤 Voice Search and Navigation
🌐 Multi-Language Support
🔊 Text-to-Speech Functionality
📱 QR Code Generation
📝 User Feedback System

# Dataset
The project uses a medicinal plant dataset containing information about various AYUSH medicinal plants, including:
Plant Name
Scientific Name
Medicinal Uses
Health Benefits
Ayurvedic Applications

# Project Architecture
User accesses the virtual herbal garden.
AI model identifies medicinal plants.
Plant information is retrieved from the database.
Users receive medicinal and wellness recommendations.
Additional features such as voice interaction and multilingual support enhance accessibility.
